1. Data folder contains all the datasets used for this assignment
Images folder contains images downloaded for comic book composition
There are subfolders for raw images based on chapters
The book folder is containing the cover and back images
There are 5 csv files contains dialogs for the specific chapter of the comic book characters

2. Program folder contains the Jupyter python notebook
comic-book.ipynb is Jupyter python notebook for composing my comic book

3. output folder contains output at each stage of the script
borderedimages folder contains images folder based on chapters with borders
textimages folder contains images folder based on chapters with captions overlaid
Imagepages folder contains horizontally stitched images
Chapters folder contains vertically stitched images
Jungle book folder contains jungleBook.pdf which is the PDF version of the comic book
